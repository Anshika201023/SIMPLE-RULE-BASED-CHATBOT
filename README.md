# SIMPLE-RULE-BASED-CHATBOT
PROJECT ON SIMPLE RULE BASED CHATBOT BASED ON HTML, CSS, JS (FRONT END) AND PYTHON (FLASK)

This is a simple rule based chatbot named ChatBuddy where questions and responses are given to the chatbot and no machine learning model is used. 

This chatbot is able to answer simple questions provided like tell a joke, greetings, fun fact, time, date, and is able to solve mathematical problems using math library. 
NLP is used to lemmatize, tokenize, stopwords and punctuation are removed from text given by user.
